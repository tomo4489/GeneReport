API一覧

  • GET /api/report-types - 登録されている報告書名の一覧を取得
  • GET /api/report/{name}/fields - 指定した報告書の項目名一覧を取得
  • POST /api/report/{name}/record - 指定した報告書へレコードを追加 
    {"field1": "value", ...}
  • POST /api/report/{name}/parse - テキストを解析してレコードを保存 
    text=... (form field)

詳細な仕様は Swagger UI でも確認できます。
